## 🧠 Overview

GrowHub is a real-time, multi-sector smart civic platform that connects emergency services, crime reporting, healthcare support, job discovery, and community news into one unified system.

It is designed to make cities safer, faster, and more intelligent using live data, maps, and AI-based prioritization.

🔗 Live Demo: https://growhub-wheat.vercel.app/dashboard

---

## 🎯 Problem Statement

Modern societies face:
- Slow emergency response systems
- Fragmented crime reporting
- Lack of real-time safety information
- Inefficient job discovery systems
- Poor coordination between citizens and services

---

## 💡 Solution

GrowHub solves these issues by creating a real-time intelligent civic ecosystem where:
- Users report emergencies instantly with GPS
- Crimes are tracked on a live map
- Jobs are discovered globally in real time
- Healthcare + ambulance + blood system is connected
- Community news is shared safely and transparently

---

## 🧩 Features & Modules

### 🚨 Emergency & Health System
- AI-based emergency prioritization (Critical → Normal)
- Blood donor matching system (location + blood group)
- Ambulance tracking system (live GPS + ETA)
- Hospital integration system
- Real-time emergency alerts

---

### 🚔 Crime Tracking System
- Live crime reporting with GPS location
- Crime categories:
  - Sexual Assault
  - Attempt to Rape
  - Physical Assault
  - Attempt to Murder
  - Theft
  - Robbery
  - Cyber Threat
  - Blackmail
  - Money Laundering
  - Physical Harassment
  - Scam Alert
  - Scam Business/Product
- Evidence upload (image/video)
- Nearby police station map integration
- “My Reports” tracking dashboard

---

### 💼 AI Job Discovery System
- Remote / Offline / Hybrid job filtering
- Global job map system
- Bangladesh job focus (for offline mode)
- Real-time job fetching from external sources
- Direct application links

---

### 🗺️ Smart Map Intelligence System
- Unified live map for:
  - Emergencies
  - Crimes
  - Jobs
  - Hospitals
  - Police stations
- Floating emergency/report button
- Location-based filtering system

---

### 📰 Community News System
- Publish crime/news instantly
- Visibility options:
  - Public
  - Friends
  - Specific users
- Upvote/downvote system
- Comment system

---

### ⚙️ Automation System
- Auto-sync reports to Google Sheets
- User Google account integration
- Real-time data logging
